Contact manager

Contact Manger It is an application that allows you to manage contacts. It implements the CRUD methods(CREATE, READ, UPDATE, DELETE).

The user can:
- Enter and add contact
- Search contacts
- Edit contact
- Delete contact

For the client side, we used the Angular framework. It is a powerful framework that implements SPA (.single page application). The communication with the API is done thanks to the HTTP protocole.

For the server side, we used the framework Spring. It is a powerful framework that lets us create amazing RESTFul API. We used the h2 database.
